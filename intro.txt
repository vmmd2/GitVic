PRUEBA CUATRO

  git status  -> Ver los cambios
  git log  -> Ver los commits
  git add -A                  ->  Sube cambios
  git commit -m "texto comit" ->  Da mensaje al cambio subido con add
  

- Ramas
  git branch ->         ver ramas
  git branch rama1    ->   crea rama1
  git checkout rama 1 -> nos mueve a la rama1
  git branch -D rama1 -> Borra una la rama1
  git checkout -b rama2 -> nos crea rama2 y nos mueve a ella
  
- Fusiones o Merges

  git checkout master   -> situarnos en la rama destino
  git merge    Test     ->  Master absorbe lo de Test
